STREETS: Every Choice Counts

A real-world life simulation and action game.

Core Idea:
A modern city where every action has consequences.
No fantasy. No reset abuse. Real life logic.

Platforms:
- PC
- Android

Engine:
- Godot Engine (Open Source)

Current Status:
- Pre-production (Design phase)

Gameplay Pillars:
1. Player freedom
2. Real consequences
3. No forced missions
4. One life, many paths

Initial Scope (Alpha):
- One city block
- One player
- One job
- One day cycle
