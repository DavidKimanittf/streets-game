# 🎮 CORE GAME SPEC (FIRST PLAYABLE VERSION)

## 🎯 Goal
Create a simple playable prototype of the game "STREETS".

The goal is NOT full realism yet.
The goal is a working loop.

---

## 🧍 PLAYER CONTROLS
- Move up, down, left, right
- Sprint (uses stamina)
- Interact button

---

## 🌍 GAME WORLD
- One small street map
- One apartment
- One shop
- One job location

---

## 🧠 NPC SYSTEM
- 1–3 NPCs only
- They walk in simple patterns
- Player can talk to them

---

## 💰 MONEY SYSTEM
- Player starts with small money
- Can earn money from one job
- Can spend money in shop

---

## ⏰ TIME SYSTEM
- Simple day cycle
- Morning → Day → Night

---

## 🔁 GAME LOOP
1. Wake up
2. Choose action
3. Earn or lose money
4. Return home
5. Sleep
6. Repeat

---

## ❌ NOT INCLUDED (YET)
- Open world
- Cars
- Complex combat
- Police system
- Large map
- Multiplayer

---

## 🧠 RULE
If it does not help the core loop, it is NOT added yet.
